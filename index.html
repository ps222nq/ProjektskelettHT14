HTML5 och SEO

Kan man använda HTML5 för att öka trafiken från sökmotorerna?

CTR - varför SEO är viktig

Hur kan HTML5 bidra?

  • Giltig, standardsenlig HTML
  • Struktur: <nav>, <footer>, <article>
  • Media utan plugins: <video>, <canvas>
  • Snabbare: färre plugins, mindre JS (med CSS3), lagring på klientsidan

Att påverka sökresultatets utseende

  • SiteLinks med <nav>
  • Rich Snippets med microdata
