CTR - varför SEO är viktig

Hur kan HTML5 bidra?

  • Giltig, standardsenlig HTML
  • Struktur: <nav>, <footer>, <article>
  • Media utan plugins: <video>, <canvas>
  • Snabbare: färre plugins, mindre JS (med CSS3), lagring på klientsidan

Att påverka sökresultatets utseende

  • SiteLinks med <nav>
  • Rich Snippets med microdata

Möjliga fördelar

  • Öka CTR
  • Minska "bounce rate"
  • Lättare att urskilja viktig data

Möjliga nackdelar

  • Mer kod
  • Kan tappa relevans

Slutsats

HTML5 kan i viss mån bidra till bättre SEO-resultat, framförallt genom snabbare sidor och bättre strukturerad innehåll. Det påverkas dock mycket av förändrande algoritmer och stödet som finns i webbläsare.
